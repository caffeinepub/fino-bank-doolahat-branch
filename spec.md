# Fino Bank Doolahat Branch – Lien Transaction Module

## Current State
The portal has modules: Dashboard, Daily P&L Entry, P&L Reports, Fixed Deposits, Transactions, Payment Heads, Merchants, Inventory, Complaints, Loans, UPI Collection. All data is persisted to localStorage. Role-based access is managed via InventoryAuthContext (staff/manager). RoleSwitcherBar is used on all key pages.

## Requested Changes (Diff)

### Add
- New `lien-transaction` tab in NavTabs and App.tsx routing
- New page `src/frontend/src/pages/LienTransaction.tsx`
- Two sections on the page:
  1. **Account Details** (top): Two-column grid
     - Left: CIF No, Customer Name, Start Date, Mobile No, Aadhar No, PAN No
     - Right: Account No, Account Type, Account Status, Balance
     - Populated from stored account profile data; visible to staff and manager
  2. **Operations** (bottom): Four sub-tabs/options
     - **Account Profile**: Form to add/update all fields from section 1. Manager-only write access; staff sees this tab but cannot edit (read-only or hidden form).
     - **Cash Deposit**: Amount input + optional remarks. On submit, records transaction to history.
     - **Cash Withdrawal**: Amount input + optional remarks. On submit, records transaction to history.
     - **Transaction History**: Table with columns: Txn Date, Reference No (autogenerated), Description, Debit, Credit, Balance. Staff can view only; manager can delete rows. Print/download as .docx (A4).

- **Reference No format**: `[last 4 digits of account no][letter A-D][credit=0/debit=1][ddmmyyyy][serial no of txn for this customer]`
  - Example for credit on 07/04/2026, acct ...1234, 3rd txn: `1234B007042026003`
  - Letter A-D is dynamic/random per transaction
  - Serial no is zero-padded to 3 digits, counting all transactions for this account

- **Print Statement (.docx)**: Button in Transaction History section that generates a Word document in A4 format containing:
  - All account profile fields at top
  - Full transaction history table
  - Uses docx library (already available via xlsx/docx pattern)

- **localStorage keys**:
  - `fino_lien_accounts`: array of account profile objects (one per CIF/account)
  - `fino_lien_transactions`: array of transaction records
